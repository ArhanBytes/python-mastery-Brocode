## 🏁 Getting Started

### Prerequisites

- [install python interpreter](https://youtu.be/ix9cRaBkVe0?si=pTrPYzf1ek80uhsn&t=47)
- [install IDE](https://youtu.be/ix9cRaBkVe0?si=pTrPYzf1ek80uhsn&t=87) - Pycharm / Visual Studio Code


### Installation
```bash
# Clone repository
git clone https://github.com/ArhanBytes/python-mastery-Brocode.git

# Navigate to project
cd python-mastery-Brocode

# Install dependencies
pip install pyqt5 requests
```

### Run Any File
```bash
# Run fundamentals lesson
python "Course_Material/01_fundamentals/01_variables.py"

# Run a project
python "projects/16_hangman/main.py"
```
