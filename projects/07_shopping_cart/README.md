# üõí Shopping Cart Program

## üìå Project Description

A simple command-line shopping cart program that allows users to add food items with prices and calculates the total cost.

---

## üöÄ What It Does

* Asks the user to enter food items
* Allows the user to enter the price for each item
* Prevents negative or zero prices
* Stores items and prices in separate lists
* Displays all items in the cart
* Calculates and prints the total price
* Stops when the user enters `q`

---

## ‚ñ∂Ô∏è How to Run It

1. Make sure Python 3 is installed.
2. Open terminal / command prompt.

    ```bash
        cd projects/07_shopping_cart
    ```
4. Run:

    ```bash
        python main.py
    ```

5. Start entering food items.
6. Type `q` to quit and see your cart summary.
