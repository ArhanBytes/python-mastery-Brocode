# 🏦 Banking System Program

A simple command-line Banking System built with Python.
This program allows users to deposit money, withdraw money, and check their balance using a menu-driven interface.

---

## 📌 What It Does

* Displays a banking menu
* Allows user to:

  * Show current balance
  * Deposit money
  * Withdraw money
  * Exit the program
* Validates user input
* Prevents negative deposits
* Prevents withdrawals greater than available balance
* Keeps running until the user chooses to exit

---

## ▶️ How To Run It

1. Make sure Python 3.10+ is installed (uses `match-case`).
2. Open terminal / command prompt.
3. Navigate to the project folder.
```bash
cd projects/13_banking_system
```
4. Run:
```bash id="t9x4mz"
python main.py
```
5. Follow the on-screen menu instructions.
